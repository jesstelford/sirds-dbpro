RENDER SIRD

This command must be called after UPDATE SIRD, otherwise no 3D data can be found to create the SIRD from.

This function actually generates the Random Dot Stereogram.
The reason this is seperated from UPDATE SIRD is to allow multiple updates of the 3D data when only one render of the SIRD is needed, etc.


Syntax

RENDER SIRD
