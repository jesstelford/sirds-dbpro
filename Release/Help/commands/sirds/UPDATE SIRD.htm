UPDATE SIRD

This command must be called before RENDER SIRD, otherwise no 3D image will be created.

This function does not generate the Random Dot Stereogram, but instead generates the internal 3D needed for it.
The reason this is seperated from RENDER SIRD is to allow multiple updates of the 3D data when only one render of the SIRD is needed, etc.

Syntax

UPDATE SIRD
